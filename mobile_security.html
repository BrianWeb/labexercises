PC vs Mobile Security Threats

Aspect PC Software Mobile Apps
Distribution Often installed from websites, USBs, or disks (uncontrolled sources). Primarily from app stores with vetting; sideloading possible.
Sandboxing Traditionally weak; many programs have system-wide access. Strong sandboxing & permissions model by default.
Update Practices Patch management inconsistent; user must install updates manually or via IT. Automatic updates via app stores (though fragmentation affects Android OS).
Malware Risk High — many decades of malware targeting PCs. Growing — malware hidden in apps, especially outside official stores.
User Behavior Users more cautious with unknown downloads. Users often grant permissions quickly, download casual apps.
Security Tools Antivirus, firewalls, IDS/IPS widely deployed. Limited security tools; OS security built-in but fewer third-party defenses.
Unique Threats Ransomware, rootkits, phishing emails. SIM swaps, malicious QR codes, sensor misuse, insecure Wi-Fi.
